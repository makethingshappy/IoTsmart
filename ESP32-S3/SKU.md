# SKU — IoTsmart ESP32-S3

```
020111vvv0eh
```

| Field | Description |
|---|---|
| `020111` | Product code — IoTsmart ESP32-S3 |
| `vvv` | Version (e.g. 201 = version 2.01) |
| `0` | Fixed |
| `e` | EYESPI connector |
| `h` | HOST connector type |

**EYESPI connector (e):**

| Value | Description |
|---|---|
| 0 | No |
| 1 | Yes |

**HOST connector type (h):**

| Value | Description |
|---|---|
| 1 | H-HOST connector |
| 2 | V-HOST connector |
